# RelicItemComandos

Crie itens que executam comandos.

## Comandos

|Comando         |Descrição                      |Permissão                    |
|----------------|-------------------------------|-----------------------------|
|/activator |Mostra a lista de itens|relicitemcomandos.admin.command    |
|/activator give |Envia um item para um player|relicitemcomandos.admin.command    |


## Download

Você pode encontrar o plugin pronto para baixar [**aqui**](https://github.com/vitinhoamaloiras/RelicItemComandos/releases).

## Dependências

### Obrigatórias

- Para que este plugin funcione, ele precisa do RelicPlugins.jar em seu servidor.
